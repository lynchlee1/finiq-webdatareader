# FnGuide Web Data Reader (FnguideTableDownloader)

FnGuide(에프앤가이드) 웹페이지(CompanyGuide)에서 기업의 재무제표 및 재무비율을 자동으로 다운로드하여 엑셀 시트에 정리하는 VBA 매크로 도구입니다.
Windows 환경에서만 작동합니다. (Mac 환경에서는 작동하지 않습니다.)

## 주요 기능
- **통합 재무 데이터 수집**: 손익계산서(IS), 재무상태표(BS), 현금흐름표(CFS), 재무비율(Ratio)을 한 번에 수집합니다.
- **연간/분기 데이터 병렬 배치**: 연간 데이터와 분기 데이터를 나란히 정렬하여 가독성을 높입니다.
- **연결/별도 재무제표 선택 지원**: 각 종목별로 연결 또는 별도(개별) 재무제표 수집 여부를 선택할 수 있습니다.
- **자동 서식 적용**: 계정 과목의 들여쓰기(Indent) 등 에프앤가이드 웹페이지의 스타일을 엑셀 시트에 반영합니다.

## 사용 방법
1. **`MAIN` 시트 구성 및 종목 코드 입력**
   - 엑셀 파일 내에 **`MAIN`** 시트를 생성합니다.
   - **B열 3행(B3)부터** 다운로드하려는 종목 코드를 입력합니다. (예: `A005930`)
   - **C열 3행(C3)부터** 재무제표 구분 값을 입력합니다. (비어있으면 기본적으로 **연결**재무제표를 수집합니다.)
     - **연결재무제표**: 빈칸, `CON`, `CONS`, `CONSOLIDATED`, `A`, `연결`
     - **별도(개별)재무제표**: `B`, `SEP`, `SEPARATE`, `STANDALONE`, `별도`

   *예시 (MAIN 시트)*
   | | A열 | B열 (종목코드) | C열 (구분) |
   |---|---|---|---|
   | **1행** | | | |
   | **2행** | | 종목코드| 구분 |
   | **3행** | | A005930 | 연결 |
   | **4행** | | A000660 | 별도 |
   | **5행** | | A035720 | |

2. **결과 확인**
   - 매크로를 실행하면 **`FnGuide`** 시트가 생성되며, 입력한 종목들의 재무 데이터가 열 방향으로 나란히 정렬되어 출력됩니다.
